# AI Doctor

AI Doctor is an AI-powered injury diagnosis and first aid guidance app.  
It helps users analyze injuries, get treatment recommendations, and access hygiene and health advice.

- Upload images OR describe your injury in text
- Get AI-based diagnosis and first aid
- Emergency instructions include Indian ambulance number 112
- Developed by Flame Coders

## How to use
1. Open the public web link (GitHub Pages)
2. Upload an injury image or describe the injury
3. Click "Analyze Injury"
4. Review recommended care steps!
